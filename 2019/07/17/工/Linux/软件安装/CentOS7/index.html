CentOS7.6 安装

下载地址：http://mirrors.aliyun.com/centos/7.6.1810/isos/x86_64/CentOS-7-x86_64-Minimal-1810.iso

1. VMware Workstation 选项

  • 新建虚拟机 ->
  • 自定义 ->
  • 稍后安装操作系统 ->
  • 命名虚拟机（例如CentOS7.6m，7.6是版本，m指最小化安装） ->
  • 自定义硬件 ->
  • 新CD/DVD中选择，使用ISO映像文件，即系统映像

2. 安装系统

  • 启动虚拟机 ->
  • Install CentOS7 ->
  • English ->
  • LANGUAGE SUPPORT 拉到下面中文，添加简体和繁体中文（前两个）支持 ->
  • INSTALLATION DESTINATION 直接左上Done，自动分区 ->
  • NETWORK & HOST NAME 右上角点 ON，这是让开机后自动联网 ->
  • DATE & TIME 选上海并打开 Network Time ->
  • 设置密码，安装完成重启

3. 自定义系统（可选）

  • 换中科大源（推荐）

  • 下载常用软件（推荐）

    yum -y install lrzsz wget gcc unzip perl

  • 关闭防火墙和selinux（推荐）

    systemctl disable firewalld
sed -i 's/SELINUX=enforcing/SELINUX=disabled/'  /etc/selinux/config

  • 安装 VMware tools

    yum -y install kernel-devel-$(uname -r)
mkdir /mnt/cdrom
mount /dev/cdrom /mnt/cdrom
cd /tmp
tar zxpf /mnt/cdrom/VMwareTools*
cd vmware-tools-distrib
./vmware-install.pl

*指的是版本编号，可以 ls /mnt/cdrom 查看

  • 添加企业附加软件包epel

    yum -y install epel-release

  • 设置静态IP

IPADDR=后即为修改的IP地址，注意网关GATEWAY地址为VMware编辑→虚拟网络编辑器→NAT设置里网关地址，一般为192.168.xxx.2
